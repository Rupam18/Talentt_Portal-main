📊 User Experience Flow

  1. Candidate completes technical test → Results calculated and displayed
  2. Score ≥ 50% → Success message with coding round eligibility badge
  3. "Go for Coding Round" button appears → Prominent green button with rocket icon
  4. Candidate clicks button → Confirmation dialog appears
  5. Candidate confirms → Eligibility data stored, email notification sent
  6. Coding round info → "Coming soon" message with next steps
